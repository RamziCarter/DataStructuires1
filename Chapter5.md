

# Stacks


* A stack is an ADT
  * LIFO structure
  * push - add element to top of the stack
  * pop - remove top element
  * peek - return the top element of the stack without removing

> If the stack is empty and you want to pop/peek: 3 options
* throw an exception
* do nothing
* return null
